## Getting started

Install Homebrew

Install RVM
Install Ruby and Rails

Install all dependencies.

```bash
$ bundle install
```

Create a Rails table.

```bash
$ rails db:create
```

Migrate the...

```bash
$ rails db:migrate
```

Seed the data.

```bash
$ rails db:seed
```

Open the Rails console and test out data.

```bash
$ rails c
```
